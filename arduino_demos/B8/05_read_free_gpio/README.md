# FeedCurrent B8 Controller Arduino Example 05: Read Free GPIO State

## Description
This example reads the state of several free GPIO pins (13,40,14,48,0,47,7,21) and prints them to the Serial Monitor.

## File Structure
- `src/B8_05_read_free_gpio.ino` – Arduino source code.
- `precompiled/B8_05_read_free_gpio.bin` – Precompiled binary.

## Using the Precompiled Binary
- Flash `precompiled/B8_05_read_free_gpio.bin` to `0x0`.

## Compiling from Source
- No extra libraries needed.
- Compile and upload.
