# Precompiled Firmware

This directory is for precompiled firmware files (.bin) for the DM32 Ethernet TCP Server example.

If precompiled firmware is available, it should be placed here and renamed to:
- `DM32_08_ethernet_tcp_server.bin`

## Purpose

The precompiled directory is a required part of the standard directory structure, even if currently empty. This ensures consistency across all Arduino examples and provides a standard location for firmware files if they become available in the future.
