# TA ESPHome Configuration

## Overview
This directory contains the ESPHome YAML configuration for the FeedCurrent TA controller, enabling seamless integration with Home Assistant.

## Files
- `TA_esphome.yaml` - Main ESPHome configuration file

## Hardware Summary
- **MCU**: ESP32-S3
- **I/O Expander**: PCF8575 @ 0x22 (12 inputs + 2 relay outputs via PCF8575)
- **Network**: WiFi (default) / W5500 Ethernet (optional, uncomment in config)
- **Tuya**: Tuya WiFi MCU integration
- **RS485**: UART1, GPIO38(RX)/GPIO39(TX)
- **1-Wire**: 12 channels on GPIO9/11/5/4/7/6/13/40/14/48/21/47

## Quick Start

### 1. Modify WiFi Credentials
```yaml
wifi:
  ssid: YOUR_WIFI_SSID
  password: 'YOUR_PASSWORD'
```

### 2. Generate New API Key
Replace the default encryption key with a new one generated by ESPHome.

### 3. Flash to Device
```bash
esphome run TA_esphome.yaml
```

### 4. Enable Ethernet (Optional)
Uncomment the `ethernet:` section and comment out `wifi:` section:
```yaml
ethernet:
  type: W5500
  clk_pin: GPIO1
  mosi_pin: GPIO2
  miso_pin: GPIO41
  cs_pin: GPIO42
  interrupt_pin: GPIO43
  reset_pin: GPIO44
```

## Tuya Integration
The configuration includes Tuya WiFi MCU support for cloud connectivity. Update `product_id` to match your registered Tuya product.

## Web Interface
The device provides a built-in web server at port 80 for direct control without Home Assistant.
