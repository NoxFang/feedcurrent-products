# ESPHome Configurations for All Products

This directory contains ESPHome YAML configurations for various controllers.

## Structure
- `B16/` - Configurations for the B16 model
- `B8M/` - Configurations for the B8M model
- `B8/` - Configurations for the B8 model
- `B4M/` - Configurations for the B4M model
- `B4/` - Configurations for the B4 model

Each subfolder contains its own README with model-specific details.
